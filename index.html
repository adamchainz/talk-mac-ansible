Project

github.com/adamchainz/mac-ansible

  • Forked or copied by many friends
  • Run every week from calendar alert

What It Does

  • Role applied to computer
  • Installation: homebrew, homebrew_cask, pip, ...
  • Settings: dotfiles symlinked from repo, or own repos, e.g ~/.pypirc
  • Crons: a few, e.g. remove temporary Python files

What It Doesn't Do

  • Installing homebrew, Python, Ansible, git
  • Configuring macOS - osx_defaults
  • Mac App Store - mas tool
  • Dotfiles passwords
